Справка по системе идентификации естественных языков

Данная система предназначена для определения языка (русский, английский) введенного текстового документа с использованием трех различных методов анализа.

1. Как пользоваться системой

  1. Ввод текста: На главной странице введите текст объемом примерно 1 страница А4 в текстовое поле.
  2. Выбор метода: Выберите один из трех доступных методов анализа (см. ниже).
  3. Получение результата: После отправки система отобразит результат идентификации (язык), добавит запись в историю и обновит сводную статистику.
Внимание: Ваши классы обработки текста автоматически очищают его от знаков препинания, цифр и лишних пробелов, переводя в нижний регистр, что позволяет работать с HTML-содержимым.

2. Методы анализа языка

1. Анализ частоты букв (AlphabetProcessor)

Принцип: Метод основан на сравнении частотного профиля букв (буквенных униграмм) во входном тексте с эталонными профилями языков (русский/английский). Для сравнения используется **Косинусное сходство** (Cosine Similarity).

  • **Метрика:** Косинусное сходство (значение от 0 до 1). Чем ближе к 1, тем выше сходство.
  • **Результат:** Выбирается язык с наибольшим сходством.

2. Анализ частоты слов (WordFrequencyProcessor)

Принцип: Метод использует профили из **наиболее частотных слов** каждого языка. Для входного текста считается сумма взвешенных частот его уникальных слов, входящих в эталонный профиль.

  • **Метрика:** Итоговый вес (Score). Сумма нормированных частот совпавших слов.
  • **Результат:** Выбирается язык, набравший максимальный итоговый вес.

3. Нейросетевой анализ (NeuroProccessor)

Принцип: Самый сложный метод, использующий обученную **нейронную сеть** для классификации текста. Текст сначала векторизуется с помощью **TF-IDF** (Term Frequency-Inverse Document Frequency), а затем подается на вход нейросети для прогнозирования класса.

  • **Метрика:** Вероятность класса (на выходе нейросети).
  • **Результат:** Выдается класс (язык) с наибольшей предсказанной вероятностью.

3. Интерпретация выходных данных

  • Активная ссылка на документ: В истории анализа отображается ссылка, имитирующая уникальный идентификатор для обработанного документа (например, /document/analysis_1634567890).
  • Язык текста: Окончательный результат идентификации, полученный выбранным методом (например, "english" или "russian").
  • Сводная статистика: Отображает общее количество проанализированных текстов и процентное распределение определенных языков в тестовой коллекции (истории анализа).

4. Сохранение и печать

  • Сохранение в файл: На главной странице доступна кнопка для скачивания файла (lang_detection_results.txt), содержащего полную историю анализа и сводную статистику.
  • Распечатка: Для распечатки результатов используйте стандартную функцию печати вашего браузера (Ctrl+P или File -> Print). Рекомендуется распечатывать страницу с результатами или предварительно сохранить их в файл.
